Scraping Data

Scraping data atau web scraping adalah teknik pengambilan data otomatis dari internet atau website menggunakan program komputer. Kita menggunakan UI Path untuk mengscrapping datanya

Cleansing Data

Cleansing data atau pembersihan data adalah proses mengidentifikasi, memperbaiki,dan menghapus kesalahan atau ketidakakuratan dalam data.

Representasi Data

Representasi data sangat penting dalam dunia teknologi informasi karena membantu memahami data dan membuat keputusan yang tepat. Contoh dari representasi data termasuk tabel, grafik, diagram, peta, dan banyak lagi.
